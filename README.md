
# onlineshop

This project is an e-commerce application with a frontend developed using Next.js, TypeScript, and Tailwind CSS. The backend is built with NestJS and uses MySQL as the database.

## Frontend
- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS

## Backend
- **Framework:** NestJS
- **Database:** MySQL
